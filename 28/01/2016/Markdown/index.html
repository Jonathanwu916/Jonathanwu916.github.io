MarkDown - Style HTML easier

Many bloggers have already known Markdown. Markdown is a way to style your HTML text. It means which can mix with HTML.

How do you learn markdown

Markdown is super easy to learn. I think it is easy for everyone to understand.To learn the syntax of Markdown, go to Mastering Markdown. If you still can not understand, try by yourself on dillinger.

What kind of Markdown Editor you recommend

For me, there are two ways I usually choose.

  • If you want to export a PDF or PowerPoint, try Madoko. Which is a online Markdown editor.It provide a powerful markdown expansion.

  • Sublime Text is my favorite text editor. Press ctrl+shift+p input install package . Then search markdown, you can see plenty plugins, choose the one you like to use.

To sum up, markdown is quite suitble for writing technology blog. Especially for the coding highlight part, which is perfectly match up displaying code.
